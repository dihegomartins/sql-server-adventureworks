# SQL Server - Estudos AdventureWorks 🚀

Repositório de exercícios práticos em T-SQL (Microsoft SQL Server) utilizando o banco de dados oficial **AdventureWorks**.

## 🧠 Conceitos Praticados

- Seleção de dados com `SELECT` e `DISTINCT`.
- Filtros condicionais com `WHERE`, `AND`, `OR`.
- Operadores avançados: `BETWEEN`, `IN`, `LIKE`.
- Ordenação e Limitação: `ORDER BY` e `TOP`.
- Agregações e Agrupamentos: `COUNT`, `SUM`, `AVG`, `MIN`, `MAX` e `GROUP BY`.
