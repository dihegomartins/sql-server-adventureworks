# SQL Server - Estudos AdventureWorks 🚀

Repositório de exercícios práticos em T-SQL (Microsoft SQL Server) utilizando o banco de dados oficial **AdventureWorks**.

## 🧠 Conceitos Praticados

- [cite_start]**Seleção de dados:** Uso de `SELECT` e `DISTINCT` para filtragem de nomes e sobrenomes únicos[cite: 3, 24].
- [cite_start]**Filtros condicionais:** Aplicação de lógica com `WHERE`, `AND`, `OR` para segmentação de dados[cite: 31, 32].
- [cite_start]**Operadores avançados:** Consultas utilizando `BETWEEN` para períodos, `IN` para listas e `LIKE` para busca de padrões de texto[cite: 24, 56].
- [cite_start]**Ordenação e Limitação:** Manipulação de resultados com `ORDER BY` e a cláusula `TOP` para rankings[cite: 24, 55].
- [cite_start]**Agregações e Agrupamentos:** Cálculos estatísticos com `COUNT`, `SUM`, `AVG`, `MIN`, `MAX` e a cláusula `GROUP BY` para análise de dados[cite: 25, 32].
- **Tipagem de Dados:** Manipulação de tipos específicos do SQL Server como `NVARCHAR(50)`, `MONEY`, `INT` e `DECIMAL(8,2)`.

## 🚀 Como Testar (Interactive Demo)

Você pode testar as queries deste repositório online, sem precisar instalar o SQL Server:

1. Acesse o [DB Fiddle](https://www.db-fiddle.com/).
2. No menu suspenso, selecione **SQL Server 2019**.
3. No painel da **esquerda (Schema SQL)**, cole o conteúdo do arquivo [`01-basics/setup.sql`](./01-basics/setup.sql).
4. No painel da **direita (Query SQL)**, cole qualquer uma das queries dos desafios presentes nas pastas.
5. Clique em **Run** para ver os resultados em tempo real.
